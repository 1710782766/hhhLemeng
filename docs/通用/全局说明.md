# 全局说明

### 请求头
Authorization=Bearer access_token
Content-type=application/json
### 返回结果
| 字段 | 说明 | 备注 |
| --- | --- | --- |
| code | 返回码 |  |
| msg | 返回码信息 |  |
| result | 返回结果 | api列表中返回结果都是基于该字段 |
| meta | 元数据(分页信息) | 部分接口没有该字段返回 |

### 全局返回码
| 返回码 | 描述 |
| --- | --- |
| 0 | 成功 |
| 10003 | 参数错误 |
| 10005 | 系统异常 |
| 20005 | 其他信息提示 |
