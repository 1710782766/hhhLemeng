# 乐檬平台接口文档（前端版）

## 统一响应格式

所有接口返回统一格式：

```json
{
  "status": 0,
  "message": "请求成功",
  "result": {
    "data": {},
    "list": []
  },
  "ext": {}
}
```

| 字段 | 说明 |
|------|------|
| `status` | 0=成功，非0=失败 |
| `message` | 提示信息 |
| `result.data` | 单条数据（对象）或分页数据 |
| `result.list` | 列表数据（数组） |
| `ext` | 扩展字段（目前为空对象） |

**数据放置规则**：
- 列表数据 → 放入 `result.list`
- 单条对象 → 放入 `result.data`
- 分页对象（含 page/total/list）→ 放入 `result.data`

## 接口列表

### 1. 商品列表查询

**路径**：`/shopItemFind.do`

**功能**：获取有库存的商品列表（纯数组，无分页信息）

**请求参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `pindex` | int | 否 | 页码，默认 1 |
| `psize` | int | 否 | 每页数量，默认 10，最大 20 |
| `item_category_code` | string | 否 | 商品类别编码，筛选指定分类下的商品（支持树形匹配，如"10"匹配"1001"、"100101"等） |

**请求示例**：

```json
{
  "pindex": 1,
  "psize": 20,
  "item_category_code": "100109"
}
```

**响应示例**：

```json
{
  "status": 0,
  "message": "获取最新消息成功",
  "result": {
    "data": {},
    "list": [
      {
        "item_num": 20439,
        "item_code": "009818",
        "item_name": "雪碧0糖0脂肪柠檬味汽水 500ml",
        "item_spec": "",
        "item_unit": "瓶",
        "item_category": "碳酸饮料",
        "item_regular_price": 3.8,
        "item_wholesale_price": 4.5,
        "item_wholesale_unit": "件",
        "item_wholesale_rate": 12,
        "item_barcode": "6954767430461",
        "item_image": "https://..."
      }
    ]
  },
  "ext": {}
}
```

**注意**：
- 只返回有库存的商品
- 商品图片已注入到 `item_image` 字段
- 乐檬 `basic.item.find/v2` 接口返回纯数组，**不包含分页信息**（无 `page`/`total` 等字段）

### 2. 商城商品查询

**路径**：`/mallItemFind.do`

**功能**：商城商品查询（乐檬新零售），返回带图片、价格、规格等完整信息的商品列表

**请求参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `pindex` | int | 否 | 页码，默认 1 |
| `psize` | int | 否 | 每页数量，默认 10，最大 200 |
| `category_ids` | array | 否 | 商品分类ID列表，如 `[1327280, 1327283]` |
| `name` | string | 否 | 商品名称模糊查询 |
| `item_name_or_code_or_barcode` | string | 否 | 商品名称/代码/条码联合查询 |
| `enable` | boolean | 否 | 是否启用，默认 `true` |
| `stop_sale` | boolean | 否 | 是否停售，默认 `false` |

**请求示例**：

```json
{
  "pindex": 1,
  "psize": 20,
  "category_ids": [1327280],
  "name": "薯片"
}
```

**响应示例**：

```json
{
  "status": 0,
  "message": "请求成功",
  "result": {
    "data": {},
    "list": [
      {
        "id": 63613,
        "name": "三迪逸品起泡网袋（2个装）",
        "description": null,
        "enable": true,
        "images": [
          {
            "id": null,
            "url": "https://mercury-image.nhsoft.cn/amazon-base/material/image/26403/9magymdy0ct1774773442566.jpg"
          }
        ],
        "category_id": 1327280,
        "brand": null,
        "basic_unit": "包",
        "shelf_life": 360,
        "code": "YEH0089851",
        "barcode": "6958417121075",
        "variants": [
          {
            "variant_id": 2640300113122,
            "variant_name": null,
            "variant_std_price": 3.5,
            "variant_lining_price": 0.0,
            "variant_level2_price": null,
            "variant_code": null,
            "variant_barcode": "6958417121075"
          }
        ],
        "min_std_price": 3.5,
        "max_std_price": 3.5,
        "sync_inventory": true,
        "safe_inventory": 0,
        "sold_amount": 0,
        "stop_sale": false
      }
    ]
  },
  "ext": {}
}
```

**响应字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | int | 商品编号 |
| `name` | string | 商品名称 |
| `description` | string | 商品描述 |
| `code` | string | 商品代码 |
| `barcode` | string | 商品条码 |
| `images` | array | 商品图片列表，每个元素包含 `id` 和 `url` |
| `category_id` | int | 商品分类ID |
| `brand` | string | 品牌名称 |
| `basic_unit` | string | 基本单位 |
| `shelf_life` | int | 保质期（天） |
| `variants` | array | 商品规格列表 |
| `variants[].variant_id` | int | 规格ID |
| `variants[].variant_std_price` | number | 标准售价 |
| `variants[].variant_lining_price` | number | 划线价 |
| `variants[].variant_level2_price` | number | 二级售价（会员价） |
| `min_std_price` | number | 最低售价 |
| `max_std_price` | number | 最高售价 |
| `sync_inventory` | boolean | 是否同步库存 |
| `safe_inventory` | int | 安全库存 |
| `sold_amount` | int | 销量 |
| `stop_sale` | boolean | 是否停售 |

**注意**：

- 此接口为**乐檬新零售**接口，与 `shopItemFind.do` 数据来源不同
- 返回数据包含完整图片信息，无需额外查询图片
- 商品价格取 `variants[0].variant_std_price` 或 `min_std_price`

### 3. 商品分类查询

**路径**：`/shopCategoryFind.do`

**功能**：获取有库存商品的分类树（只返回有库存商品的分类，包含其父分类）

**请求参数**：无

**请求示例**：

```json
{}
```

**响应示例**：

```json
{
  "status": 0,
  "message": "获取最新消息成功",
  "result": {
    "data": {},
    "list": [
      {
        "category_code": "1001",
        "category_name": "饮料",
        "item_count": 10
      },
      {
        "category_code": "100109",
        "category_name": "碳酸饮料",
        "parent_category_code": "1001",
        "item_count": 5
      }
    ]
  },
  "ext": {}
}
```

**响应字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `category_code` | string | 分类编码 |
| `category_name` | string | 分类名称 |
| `parent_category_code` | string | 父分类编码（可能为空） |
| `item_count` | int | 该分类下有库存的商品数量 |

**注意**：
- 只返回有库存商品的分类
- 会保留有商品分类的所有祖先分类（确保分类树完整）
- 返回数据中包含 `item_count` 字段，表示该分类下有库存的商品数量

### 5. 创建批发订单

**路径**：`/wholesaleOrderCreate.do`

**功能**：提交购物车商品生成批发预约单（自动审核）

**请求参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `items` | array | 是 | 商品列表 |
| `items[].item_num` | string | 是 | 商品编码 |
| `items[].item_use_qty` | number | 是 | 数量（使用基本单位数量） |
| `items[].item_use_unit` | string | 是 | 单位（使用基本单位，如"瓶"） |
| `memo` | string | 否 | 订单备注 |
| `address_id` | string | 否 | 收货地址ID（不传使用默认客户） |

**请求示例**：

```json
{
  "items": [
    {
      "item_num": "20439",
      "item_use_qty": 24,
      "item_use_unit": "瓶"
    },
    {
      "item_num": "20440",
      "item_use_qty": 12,
      "item_use_unit": "瓶"
    }
  ],
  "memo": "客户要求尽快发货",
  "address_id": "3901383351133131"
}
```

**响应示例**：

```json
{
  "status": 0,
  "message": "请求成功",
  "result": {
    "data": {
      "wholesale_book_fid": "202603240001",
      "wholesale_book_out_bill_no": "EX2026032412345678",
      "state_code": 2,
      "state_name": "已审核"
    },
    "list": []
  },
  "ext": {}
}
```

**注意**：
- 使用 `item_use_qty` 和 `item_use_unit`（基本单位）而非批发单位
- 单价由系统根据客户和商品自动获取
- 创建的订单默认就是"已审核"状态（`state_code=2`）

**功能**：分页查询批发订单列表

|------|------|------|------|
| `pindex` | int | 否 | 页码，默认 1 |
| `psize` | int | 否 | 每页数量，默认 10 |
| `start_date` | string | 是 | 开始日期，格式 `yyyy-MM-dd` |
| `end_date` | string | 是 | 结束日期，格式 `yyyy-MM-dd` |

      {
        "wholesale_book_fid": "202603240001",

{
  "status": 0,

## 重要注意事项

### 1. 商品单位换算

- **基本单位**（`item_unit`）：如"瓶"、"个"
- **批发单位**（`item_wholesale_unit`）：如"件"
- **换算比例**（`item_wholesale_rate`）：1 件 = 12 瓶

**下单时使用基本单位**：

- 数量（`item_use_qty`）：按基本单位填写（如 24 瓶）
- 单位（`item_use_unit`）：使用基本单位（如"瓶"）
- 单价由系统根据客户和商品自动获取

### 2. 订单查询时间限制

批发订单查询（`wholesaleOrderFind.do`）**时间间隔不能超过 5 天**，建议：

- 默认查询最近 3 天的订单
- 提供日期选择器限制范围
- 如需查更早的订单，分多次调用

### 3. 订单状态说明

| 状态码 | 状态名 | 说明 |
|--------|--------|------|
| 1 | 制单 | 草稿状态，可删除 |
| 2 | 已审核 | 正式订单，管理端可见 |
| 3 | 已过期 | 超过交货期限 |
| 7 | 中止 | 已中止的订单 |
| 9 | 已作废 | 已取消的订单 |

**注意**：
- 当前创建订单接口是 `saveandaudit`（新增并审核），所以创建的订单默认就是"已审核"状态（`state_code=2`），供应商在管理系统中应该能看到
- 删除订单只能删除"制单"状态（`state_code=1`）的订单

### 4. 外部流水号

每个订单会自动生成外部流水号（`wholesale_book_out_bill_no`），格式为 `EX` + 时间戳 + 随机数，用于：

- 与第三方系统对账
- 问题排查时提供给供应商

### 5. 固定配置说明

以下配置已在服务端固定，前端无需传递：

| 配置项 | 值 | 说明 |
|--------|-----|------|
| `branch_num` | `99` | 门店编码 |
| `storehouse_num` | `264030025` | 仓库编码 |
| 默认 `client_fid` | `0026403010000001` | 好惠花默认客户编码（不传 address_id 时使用） |
| `operator` | `好惠花1` | 订单操作员 |

**关于收货地址**：

- 每个收货地址对应乐檬系统中的一个客户（`client_fid`）
- 下单时传入 `address_id` 使用对应地址的客户，不传则使用默认客户
- 地址数据在本地维护，通过 `addressList.do` 和 `addressCreate.do` 管理

### 6. 购物车维护

购物车数据由**前端自行维护**（如 LocalStorage），服务端只负责最终的下单提交。建议购物车数据结构：

```javascript
{
  "items": [
    {
      "item_num": "20439",
      "item_name": "雪碧0糖0脂肪柠檬味汽水 500ml",
      "item_use_qty": 24,
      "item_use_unit": "瓶",
      "item_image": "..."
    }
  ],
  "total_count": 24,
  "total_amount": 9.0
}
```

### 7. 错误处理

当接口返回错误时：

```json
{
  "status": 2,
  "message": "商品编码不能为空",
  "result": {
    "data": {},
    "list": []
  },
  "ext": {}
}
```

前端应：

- 检查 `status` 非 0 时表示错误
- 使用 `message` 字段向用户展示错误信息
- 网络错误时统一提示"网络异常，请稍后重试"

## 附录：服务端已封装的批发相关接口

以下接口由服务端封装，前端通过上述 HTTP 接口间接调用：

| 接口方法 | 功能 | 文档地址 |
|----------|------|----------|
| `nhsoft.amazon.wholesale.book.find` | 批发订单查询（支持分页、日期范围） | [文档](https://console.nhsoft.cn/pages/func/api-manage/122/146/nhsoft.amazon.wholesale.book.find) |
| `nhsoft.amazon.wholesale.book.read` | 批发订单读取（支持按订单号查详情） | [文档](https://console.nhsoft.cn/documents/api-doc/122/146/nhsoft.amazon.wholesale.book.read) |
| `nhsoft.amazon.wholesale.book.save` | 批发订单新增 | [文档](https://console.nhsoft.cn/documents/api-doc/122/146/nhsoft.amazon.wholesale.book.save) |
| `nhsoft.amazon.wholesale.book.saveandaudit` | 批发订单新增并审核 | [文档](https://console.nhsoft.cn/documents/api-doc/122/146/nhsoft.amazon.wholesale.book.saveandaudit) |
| `nhsoft.amazon.wholesale.book.delete` | 批发订单删除 | [文档](https://console.nhsoft.cn/documents/api-doc/122/146/nhsoft.amazon.wholesale.book.delete) |

### 批发订单查询 vs 读取的区别

| 能力 | `find` 查询 | `read` 读取 |
|------|------------|-------------|
| 主要用途 | 列表查询 | 单条详情查询 |
| 查询条件 | 日期范围、客户编号 | 订单号 或 外部流水号 |
| 时间限制 | 日期间隔 ≤ 5 天 | 无限制 |
| 返回数据 | 纯数组（无分页） | 单条（含完整明细） |
